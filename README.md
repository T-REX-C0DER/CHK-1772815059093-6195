# HelpSphere

HelpSphere is a transparent social impact platform that connects people with verified NGOs to donate, volunteer, and create measurable real-world impact.

## Features

- NGO Verification System
- Transparent Donation Tracking
- Volunteer Opportunities
- Social Impact Stories
- Shelter Requests for homeless, elderly and orphans
- AI powered recommendations
- Gamified user profiles (ranks & badges)

## Tech Stack

Frontend
- Next.js
- React
- TypeScript
- TailwindCSS

Backend
- Node.js
- MongoDB

AI
- AI impact insights
- Recommendation engine

cd helpsphere-platform
npm install
npm run dev
